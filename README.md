## Melhorias aplicadas
  - Alteração no arquivo de Logo, saindo de PNG para WEBP.
  - Alteração na cor do texto em Count, saindo de cinza para uma cor mais próxima do preto, deixando mais acessível: 
          .count__text {
            font-weight: bold;
            color: #1d1d1d;
          }
  - Alteração no Header, colocando a imagem da logo como Priority.
  - Alteração no PokemonCard, deixando as imagens em estado loading="lazy".
